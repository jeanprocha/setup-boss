# What Belongs Here

- tooling requirements (Node.js version, npm)
- setup steps and commands
- environment variable guidance (non-sensitive)

- runtime access patterns (API key config)

- secrets, tokens, or credentials
- production infrastructure details
- deploy targets or SSH hosts
